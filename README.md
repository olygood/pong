## Difficulté du jeu  :

- rebond sur les 4 murs
    - quatre direction X,Y , -X,Y , X,-Y , -X, -Y
- rebond sur les pads
- détection quand on est passer les pad
- quand on a laisser passer la balle :
    - elle va au centre
    - et retourne vers vous
